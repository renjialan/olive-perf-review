WishDesk is the full-stack internal operations platform (React + Express.js + TypeScript + MySQL) powering the entire business — proposals, recipients, AI-driven gift selection, eCard design, surveys, promotional codes, and more. AI, Agentic Systems & MCP Infrastructure

WishDesk runs a production-grade AI infrastructure stack that Olive has built features on top of and contributed to throughout the review period. The system combines LLM orchestration, vector search (RAG), real-time tool execution, and the Model Context Protocol (MCP) — a standard for connecting AI models to external tools and data.

MCP MCP Server & Tool Platform

The WishDesk MCP server exposes 50+ tools over HTTP via JSON-RPC 2.0, enabling AI models (Claude, ChatGPT) and internal workflows to interact with the platform programmatically — creating proposals, looking up orders, rendering eCards, searching the knowledge base, and more.
  • 50+ MCP tools covering proposals, eCards, orders, products, contacts, logos, templates, KB search, and session management
  • Dual authentication: internal (localhost, no auth) and external (API key via nginx proxy)
  • Session management with 50 concurrent session limit, 5-minute timeout, and tiered memory cleanup (400/600/800 MB thresholds)
  • Tool whitelist for external access — ChatGPT integrations get a curated 15-tool subset
  • Prometheus metrics, health checks, and performance monitoring endpoints
  • Deployed as a separate PM2 process alongside the main Express server

AI Gift Genie Conversational AI

The Gift Genie uses OpenAI function calling in a multi-turn agent loop to guide users through gift selection. The AI calls MCP tools (product catalog search, template rendering, proposal creation) based on conversation context, creating a fully autonomous gift-ordering experience.
  • OpenAI function calling with MCP tools as callable functions
  • Multi-turn conversation with state management and session persistence
  • Tool chaining: AI searches products, refines results, collects info, creates proposals, and returns confirmation
  • Database-backed prompt management (no hardcoded prompts) for security
  • Guest completion flow with identity resolution and metadata batching
  • Retool webhook integration for downstream processing

RAG Vector Search & Knowledge Retrieval

A Retrieval Augmented Generation (RAG) pipeline using Qdrant vector database and OpenAI embeddings to give AI responses grounded in company knowledge, admin instructions, and historical agent interactions.
  • Qdrant vector database with 4 collections: knowledge base (kb-v2), admin instructions, agent chat history, draft instructions
  • OpenAI embeddings for semantic search across all collections
  • Article sync service for automated knowledge base ingestion
  • Graceful degradation — chatbot runs without context if Qdrant unavailable
  • Connection retry logic with exponential backoff

Workflow SWIM Workflow Engine

SWIM (Smart Workflow Intelligence Manager) is a declarative workflow orchestration system that chains AI prompts, tool calls, and custom logic into automated business processes — email classification, draft generation, agent suggestions, and more.
  • JSON-defined workflows with 4 step types: prompt, tool, validation, custom
  • Variable resolution system with dot-notation and step output references
  • Email processing pipeline: classification → context gathering (parallel) → draft generation → branching → session update
  • WebSocket API for real-time workflow execution and progress streaming
  • Queue monitor service for automated background processing
  • Retry logic, timeout handling, and error recovery patterns

AI Agentic Chat & Email Automation

An autonomous chat agent that uses OpenAI function calling to handle customer support inquiries — looking up orders, checking knowledge bases, and drafting responses with minimal human intervention.
  • Function calling agent loop: call OpenAI → parse tool_calls → execute tools → return results → repeat until done
  • Context-aware responses synthesized from KB articles, admin instructions, and order history
  • Email classification using OpenAI for intent detection and response routing
  • AI-generated draft responses with quality validation
  • Agent suggestion generation for human operators
  • AI order/ticket summaries for the ops team

Vision Image Analysis & Media AI

AI-powered image analysis for the design pipeline — categorizing uploaded designs, assessing quality, and automating the mug customization workflow.
  • Google Generative AI for image analysis and design assessment
  • Replicate for ML model inference and video/image generation
  • Google Veo integration for video eCard generation
  • Batch mug image rerendering with AI-assisted processing
  • Automated design categorization and quality scoring
